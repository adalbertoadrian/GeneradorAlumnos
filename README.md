# Descripción
Script Generador de Alumnos.
## Configuración
Editar Archivo config.json y agregar credenciales de la base de datos.
```bash
npm install
nodejs createTables.js
```
Se creara toda la estructura de la base de datos.
## Ejecutar
```bash
nodejs index.js
```
